Stickerizer

Stickerizer allows someone on a UNIX based filesystem to create Telegram messenger compatible stickers from a folder of image files. I did this as a Ruby rewrite of a bash script I wrote some time ago, though ironically the rewrite is both less lightweight and slower(but the code is of higher quality).

Vernacular: Openbox3 Theme

I have written and modified existing themes for the old school Openbox window manager. The GTK (Gnome Toolkit) often used in Linux desktops uses CSS heavily. My theme is based on an old ArchLabs Linux theme, and is called vernacular.
